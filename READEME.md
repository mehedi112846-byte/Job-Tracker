1.Difference between getElementById, getElementByClassName, querySelector, querySelectorAll?
Answer: 
getElementById: This can select one unique element from any section,div or somewhere like this.
getElementByClassName: This can select and make a list of all elements of a class.
querySelector: This can find out the first element by css selector.
querySelectorAll: This can find and make an element list by css selector.

2.How do you create and insert a new element into the DOM?
Answer: 
For creating new element: 'document.createElement('tagName')'
For showing that on page: 'appendChild()' or 'insertBefore()' [by this we can get in this in a parent element]

3.What is event Bubbling?
Answer:
event Bubbling is that kind of process where the element can move towards parent that means to upper part by clicking on the child.

4.What is event Delegation?
Answer:
The process of Keeping a public listener to the parent element instead of child element. It can reduce memory space.

5.Difference between preventDefault() and stopPropagation()?
Answer:
preventDefault(): It can control abnormal behavior like clicking on link make a load on page or something like this.
stopPropagation(): It can control events to go to the parent or make bubbling.
